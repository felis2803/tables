# Experiment Checklist

Before a new run:

1. Pin the input file.
2. Pin the pipeline step list.
3. Pin all numeric parameters.
4. Choose a run id and output location.

After a run:

1. Save final tables.
2. Save all reports.
3. Record `tables / bits / rows`.
4. Compare against the previous best result.
5. Write a short summary of what changed and why it matters.
